## Workflow на 1 час

1. Открой Telegram по ссылке (или t.me/handle в браузере)
2. Группа существует? Не channel? Member count норм?
3. Тап на название → **Administrators** — увидишь @handle админа
4. Если admin не виден — открой pinned message, обычно там контакт админа
5. DM админу шаблон ниже
6. Помечай результат в этой таблице (✅ согласился / ❌ отказ / ⏳ ждём ответ)

про DMV в иммигрантских чатах постоянные.

## Better data (если хочешь свежие цифры)

Платформы для верификации/поиска ещё групп:
- **tgstat.com / tgstat.ru** — поиск "США", "USA", "Miami", "Latinos" + sort by members
- **telemetr.io** — analytics + admin contacts часто видны
- **combot.org/top/telegram/groups** — top by region/language
- **tlgrm.eu/channels** — category tags

5 минут на tgstat.com = в 5 раз больше верифицированных групп с актуальными цифрами.

## Метрика для test-run

**Цель недели:** 10 outreach DM → измерить response rate.
- Если 3+ ответов / 1+ "да" → канал работает, масштабируем
- Если 0 ответов / все молчат → канал мёртв, переключаемся на Reddit/Instagram

Не надо больше 10 DM в неделю — выглядит как массовый спам, и админы между собой переговариваются.
